# UI 视觉复刻实现状态（2026-06-26）

## 1. 本轮确认的验收边界

按用户最新要求，`doc/04_assets/ui_suite_gpt_v1/screens/pages/*.png` 只能作为视觉前端模板和验收标尺，不能把 UI 图复制到 `web/ui` 后作为页面覆盖层、整页背景图或静态替代页面。

前端开发必须满足以下边界：

- 以 React 组件、DOM、CSS 和真实交互开发成 UI 图的样子，不允许用整页 PNG 覆盖页面。
- 页面含多个 Tab 或页面内模式时，按同一个页面组件开发多个 Tab 状态；每个 Tab 对应仓库内已有 UI 图进行 1:1 对照。
- 保留真实路由、鉴权、API 适配、Tab 切换、表格、表单、按钮、告警、证据区等业务实现。
- K8s/APISIX 是最终运行底座；前端页面测试使用 Codex Desktop 内置浏览器。
