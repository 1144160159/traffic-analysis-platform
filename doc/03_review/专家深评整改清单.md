# 专家深评整改清单

更新时间：2026-06-18  
用途：替代“只计数不够深”的评审台账，作为后续产品设计、技术设计、测试验收和售前材料的整改入口。

## 0. 评审修正说明

此前 `multi_role_review_ledger.csv` 解决的是“评审规模可计数”，但不能单独代表“专家评审质量”。本清单补足高密度深评，采用“问题 -> 影响 -> 整改动作 -> 验收证据 -> 责任角色”的格式，并按真实项目角色拆分。后续应以本清单驱动 PRD、SDD、测试计划和试点交付材料的修订。
